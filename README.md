# Andrew's Infrastructure
Infrastructure for Andrew's services.

## Setup
Install [Pulumi](https://www.pulumi.com/docs/get-started/install/).

## Development
1. Make changes.
2. Preview changes.
    ```
    cd pulumi
    pulumi preview
    ```
3. Create PR.
4. Merge PR.
